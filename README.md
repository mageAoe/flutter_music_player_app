# flutter_music_player_app

### 插件

- 图片缓存CachedNetworkImage实现图片持久化到本地，Flutter默认缓存到内存，但不持久化。
- 音乐播放：audioplayer
- 视频播放：video_player (底层原理是各平台实现对应播放器，例如Android端使用EXOPlayer)
- 网络连接检测：connectivity
- 横竖屏切换：orientation


### 目录结构: 

- custom_drawer 侧边栏
- navigation_home_screen 首页导航
